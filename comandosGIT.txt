1 - PARA CONECTAR UM REPOSITORIO LOCAL COM O GITHUB

git init
git add .
ou
git add ./arquivo.extensão
git commit -m "projeto inicial"
git branch -M main
git remote add origin git@github.com:SEU_USUARIO/SEU_REPOSITORIO
git push -u origin main

2 - BAIXAR UM REPOSITORIO DO GITHUB

git clone url -> copia um repositorio do github para o seu dispositivo

3 - BAIXAR ALTERAÇÕES FEITAS NO PROJETO POR TERCEIROS NO PROJETO

git pull origin main

4 - VARIADOS

* git status -> ver os arquivos que foram alterados

* git log -> ver os commites que foram feitos

* git revert <ID do comit> -> reverte as alterações feitas por um comit

* git reset --hard <ID do comit anterior> -> apaga o comit

* git commit --amend - m "nova mensagem" -> alterar a mensagem do commit anterior
